WhatsApp-группы

Включение означает, что сервер сам сверяет ссылки вида chat.whatsapp.com из карточек компаний с группами, которые отдаёт WhatsApp-шлюз. Интервал задаёт паузу между реальными обходами: система проверяет каждого настроенного WhatsApp-клиента один раз за запуск, а не каждую компанию отдельно.

Работает только если шлюз поддерживает GET /groups и возвращает groupId вместе с invite-ссылкой или invite-code. Если шлюз этого не умеет, старый сценарий не ломается: компания привяжется после входящего сообщения из нужной группы.

Общие чаты компаний

Если несколько компаний используют одну и ту же ссылку чата, система скопирует уже найденные WhatsApp, Telegram и MAX id из привязанной компании в остальные компании с этой же ссылкой.

Существующие отличающиеся id не перезаписываются: такие группы будут отмечены как конфликтные.

Значения применяются сразу после сохранения. Дефолт расписания сохранён как раньше: 11:30 утром и 22:00 вечером. Если поставить 0 минут для выгула, специалист сможет делать выгул без паузы.
